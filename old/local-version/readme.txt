ATTENTION: Disable any dark mode extentions in your browser

Open "config.json" with notepad (or similiar) 
Insert the name, email, etc you want to show in the signature
ATTENTION:
|-- When editing the values, don't remove the quotation marks (") before and after the value you insert in.
|-- Anyting you write inside quotation marks will be what is displayed in your signiture.
|-- This includes any spaces or other characters.

Go to your Outlook desktop app

Create a new signature

Open "index.html" in a web browser and copy all contents in the webpage (CTRL + A, CTRL + C)
Paste the contents copied in the new signiture

Verify that all the fields are correctly filled!
Save the signiture and enable it

Done!
